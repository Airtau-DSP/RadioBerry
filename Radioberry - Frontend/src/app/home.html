The Radioberry, a nice small standalone amateur HF transceiver

The radioberry is a radio card connected to a raspberrypi 2 or 3

The backend uses a server.

This makes it possible to create a specific frontend. The frontend presented here uses a browser using some 'modern' frontend techniques.


The available widgets can be found on the demo widgets page; these widgets are actually controlling the radio.

Demo radio control widgets
